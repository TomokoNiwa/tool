# 業務補助ツール一式 (Productivity Tools)

本リポジトリは、実務における業務効率化を目的に制作したツール群を公開・管理するための場所です。

##ツール構成
現在、以下のツールを公開しています。

### 1. 記事制作ポータル (`editor/`)
入稿作業の効率化を目的としたエディター群です。
- **エディター**: 自動インデント（字下げ）機能付きのHTML生成。
- **画像タグ生成**: ID入力によるレイアウトコードの一括作成。

### 2. 進行管理プランナー (`planner/`)
プロジェクトの進行を補助するツール群です。
- **スケジュール管理**: 公開スケジュールの可視化。
- **タスク管理**: 業務タスクのステータス管理。
