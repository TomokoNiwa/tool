# 業務補助ツール一式（引き継ぎ用）

本リポジトリは、記事制作および進行管理の業務効率化を目的に作成された補助ツール群を管理するものです。

## 📁 ツール構成
本リポジトリには以下のツールが含まれています。

- **[記事制作ポータル (index.html)](index.html)**: 全ツールの入り口ページ。
- **editor/**: 記事制作エディターおよび画像タグ生成ツール。
- **planner/**: スケジュール管理およびタスク管理ツール。
- **zip/**: 各ツールのバックアップ用アーカイブ。

## 🚀 利用方法
GitHub Pagesにてホスティングされています。
ブラウザから以下のURLにアクセスして利用してください。
（URL：`https://[あなたのユーザー名].github.io/tool/`）

## 🛠 メンテナンス・管理について
- **行数の変更**: 各ツールの入力行数を変更したい場合は、`js/portal-tools.js` 等の初期化関数内（`initForms`など）の数値を直接書き換えてください。
- **ルールの調整**: エディターの字下げルールなどは、各ツールのJavaScript内の `update` 関数を修正することで調整可能です。

## ⚠️ 免責事項
- **現状渡し**: 本ツールは「現状渡し」の補助ツールです。制作者の離職に伴い、今後のアップデートやバグ修正、ブラウザの仕様変更に伴うメンテナンス等は行われません。
- **確認の徹底**: 生成されたHTMLコードやタスクの数値等は、必ず利用者が目視で最終確認を行ってください。
- **責任の制限**: 本ツールの利用により生じた直接的・間接的な不具合、損害、またはトラブルについて、制作者は一切の責任を負いかねます。
